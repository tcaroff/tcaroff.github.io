Overview

A system that allows to apply custom gravity forces to objects.

UE4, C++
Made alone
Summer 2018

This project is part of a bigger project that I want to achieve. This is a very fast moving system for TPS that would give the character abilities similar to manga's heroes: running on the walls, jumping very far and so on. I have extended the first part of the project so that I can assign independent attraction forces to each object so that the characters can run on the walls. But it is much more powerful than a system limited to this function.
