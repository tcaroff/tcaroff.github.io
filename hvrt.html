Tool overview

A VR tool set for professional interior designer.

Unity (first-contact), C#, HTC Vive, NVIDIA Ansel
Made alone
12 full-time weeks
I worked on: all aspects

The Hubstairs VR Tool is a tool set for interior design. It has been designed for professional use inside Hubstairs, but it also had to be able to be used by with a lot of fun and ease for other commercial uses. The developed tools allow the user to move, to change the paints and the geometry of the room, to furnish the room, to manipulate furnitures, modify and precisely position them with functionalities likes the group tool, and to generate beautiful shots for clients.
Accessibility tools for people with reduced mobility have also been added.
